# Target Brazil E-Commerce Analysis using SQL and BigQuery

## Project Overview

This project presents an end-to-end business analysis of Target Brazil's e-commerce operations using SQL and Google BigQuery.

The analysis focuses on customer behavior, order trends, freight costs, delivery performance, and payment preferences to identify opportunities for improving profitability, operational efficiency, and customer experience.

## Business Questions Answered

- Order growth trends over time
- Monthly seasonality analysis
- Customer distribution across states
- Freight cost analysis
- Delivery performance analysis
- Payment behavior analysis
- Installment usage analysis
- State-wise business performance comparison

---

## Key Findings

- Order volume increased from **329 orders in 2016** to **54,011 orders in 2018**.
- Customer demand is heavily concentrated in southeastern Brazil.
- Freight costs and delivery times vary significantly across states.
- Several states consistently receive orders earlier than estimated delivery dates.
- Credit cards dominate payment transactions.
- Installment payments are widely used by customers.

---

## Strategic Recommendations

- Establish regional fulfillment centers to reduce freight costs.
- Improve delivery estimates in fast-performing regions.
- Adopt region-specific growth strategies.
- Optimize payment options and installment offerings.
- Align marketing campaigns with customer purchasing behavior.
